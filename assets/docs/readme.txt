README.TXT for "iSilo(TM) for Android" version 6.10 (3/1/13)
------------------------------------------------------------
The iSilo program file serves as both the 30-day trial version and the registered version of the software.  If you would like to continue to use iSilo beyond the 30-day trial period, you can find information on purchasing from <http://www.iSilo.com>.  After you make your purchase, you receive a license registration code that you enter into iSilo to convert its status to registered.

Be sure to read through the manual for answers to questions you may have about the usage of features within the application.

For frequently asked questions and knowledge base articles, please visit the support page of the iSilo website at <http://www.iSilo.com>.

Please also visit the iSilo forum at <http://forum.iSilo.com>.  It's a place where you can get support, be part of a community, and voice your opinions.  To post messages at the forum, you need to become a registered forum member first by clicking the Register link there and following the instructions.  Registration at the forum is free.

If you would like to create your own documents for iSilo, you can use iSiloX.  More information about iSiloX can be found at its website at <http://www.iSiloX.com>.
